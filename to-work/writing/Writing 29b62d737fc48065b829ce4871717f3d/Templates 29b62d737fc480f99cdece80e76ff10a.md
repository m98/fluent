# Templates

```markdown
Beste meneer/mevrouw [LAST NAME],

Ik schrijf u omdat [WHY YOU'RE WRITING - 1 sentence].

[EXPLAIN SITUATION - 1-2 sentences]

Zou het mogelijk zijn om [YOUR REQUEST/QUESTION]?

Ik hoor graag van u.

Met vriendelijke groet,
[YOUR FULL NAME]
```

#### **Real Example Using This Template:**

**Scenario:** Ask your boss (meneer Jansen) for Friday off because of a doctor appointment.
```
Beste meneer Jansen,

Ik schrijf u omdat ik volgende week vrijdag vrij wil vragen.

Ik heb een afspraak bij de dokter om 10:00 uur. De afspraak duurt ongeveer twee uur.

Zou het mogelijk zijn om vrijdag 5 november vrij te hebben?

Ik hoor graag van u.

Met vriendelijke groet,
[Your name]
```

**See? Same structure, just filled in the blanks!**

### **TEMPLATE 2: Informal Email**
(Use for: colleagues, classmates, friends)
```
Hallo [FIRST NAME],

[WHY YOU'RE WRITING - 1 sentence]

[EXPLAIN SITUATION/PROBLEM - 1-2 sentences]

Kunnen we [YOUR SUGGESTION/QUESTION]?

Laat je het me weten?

Groetjes,
[YOUR FIRST NAME]
```

#### **Real Example Using This Template:**

**Scenario:** Tell your classmate Amber you can't meet tomorrow, suggest meeting next Tuesday instead.
```
Hallo Amber,

Ik kan morgen helaas niet afspreken.

Ik moet naar de tandarts en dat duurt de hele ochtend. Het spijt me!

Kunnen we volgende week dinsdag om 14:00 uur afspreken?

Laat je het me weten?

Groetjes,
[Your name]
```

### **TEMPLATE 3: Newsletter/Wijkkrant Text**
(Use for: writing about yourself)
```
[SENTENCE 1: State what you do/like]

[SENTENCE 2: Add details - when/where/with whom]

[SENTENCE 3: Explain why/how you feel about it]
```

#### **Real Example Using This Template:**

**Scenario:** Write about your weekend for the neighborhood newsletter.
```
Ik wandel graag in het weekend.

Ik doe dit meestal op zaterdag met mijn vrouw in het park.

Dit is ontspannend en gezond voor mij.
```

**Or another example - your favorite celebration:**
```
Ik vier elk jaar mijn verjaardag met vrienden.

We gaan altijd uit eten in een restaurant en daarna drinken we wat.

Dit is gezellig omdat ik graag tijd met mijn vrienden doorbreng.
```

### **TEMPLATE 4: Form Filling**
(Use for: registration forms, complaint forms, insurance forms)

This isn't really a "template" but a **strategy**:

#### **Part A: Personal Information Section** (Always the same)
```
Voor- en achternaam: [Your full name]
Adres: [Street name + number]
Postcode: [Postcode]
Woonplaats: [City]
Telefoonnummer: [Phone]
E-mailadres: [Email]
Geboortedatum: [DD-MM-YYYY]
```

**ğŸ’¡ Action:** Memorize YOUR real details in Dutch spelling by Day 3.

#### **Part B: Description Section** (Use complete sentences)

**Strategy:** Turn the pictures/prompts into simple sentences.

**Example - Insurance damage form:**
```
De dieven hebben mijn laptop gestolen.
Ze hebben ook mijn horloge meegenomen.
Het raam van de woonkamer is kapot.
```

**Example - Municipal complaint form:**
```
Er ligt veel vuilnis op straat.
Er is een stoeptegel los.
```

### **TEMPLATE 5: Work Note/Message**
(Use for: leaving instructions for a colleague)
```
Hallo [NAME],

Kun je vandaag drie dingen doen?

[TASK 1 - 1 sentence]
[TASK 2 - 1 sentence]  
[TASK 3 - 1 sentence]

Alvast bedankt!

Groetjes,
[YOUR NAME]
```

#### **Real Example Using This Template:**

**Scenario:** Leave note for colleague Fariha about tasks (based on pictures: vacuum, hang clothes, lock door)
```
Hallo Fariha,

Kun je vandaag drie dingen doen?

Wil je de winkel stofzuigen?
Kun je ook de kleren ophangen?
Vergeet niet om de deur op slot te doen aan het eind van de dag.

Alvast bedankt!

Groetjes,
[Your name]
```
