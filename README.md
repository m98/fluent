### Evidence-Based Methods

This system implements proven learning science:

**1. Active Recall** - You retrieve from memory before seeing answers
- ğŸ”¬ Research shows: **2-3x better retention** than passive review

- ğŸ”¬ Research shows: **Prevents forgetting curve**, optimizes long-term retention

**3. Immediate Feedback** - Mistakes corrected within seconds
- ğŸ”¬ Research shows: **Faster learning** when feedback is immediate

- ğŸ”¬ Research shows: **Better discrimination** and long-term retention
